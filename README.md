# Atividade de Programação 1 com Bootstrap 5

## 📋 Descrição

Esta atividade teve como objetivo praticar a utilização do Bootstrap 5 para a criação de formulários web responsivos e a manipulação de dados enviados pelo usuário.

O projeto foi desenvolvido a partir de um exemplo de formulário disponibilizado na documentação do Bootstrap 5, recebendo adaptações para atender aos requisitos da atividade.

---

## 🚀 Modificações Realizadas

### 1. Adição do campo CPF

Foi adicionado ao formulário original um novo campo para coleta do CPF do usuário.

**Campo incluído:**
- CPF

Essa alteração permitiu ampliar as informações coletadas pelo formulário.

---

### 2. Criação da página de sucesso

Foi criada uma segunda página chamada **success.html**.

Essa página é responsável por:

- Receber os dados enviados pelo formulário;
- Exibir as informações preenchidas pelo usuário;
- Confirmar o envio dos dados de forma visual e organizada.
